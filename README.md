# Clean Architecture with ASP.NET Minimal APIs

Proof-of-concept of the [Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html) utilizing ASP.NET 7 [Minimal APIs](https://learn.microsoft.com/en-us/aspnet/core/fundamentals/minimal-apis/overview).

A couple of useful articles by Herberto Graça:

- [The Entity-Boundary-Interactor (EBI) Architecture](https://medium.com/the-software-architecture-chronicles/ebi-architecture-eb956122b5cf)
- [Clean Architecture: Standing on the shoulders of giants](https://medium.com/the-software-architecture-chronicles/clean-architecture-standing-on-the-shoulders-of-giants-168e6b7b00b9)
